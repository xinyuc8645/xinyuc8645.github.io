Challenges

To be honest, there wasn't any too difficult challenge I faced for this project, but if I had to say one it probably had to do with paragraphs not allinging and images proportions. The reason why I am saynig this is because, if you look at my website for the seciton Part A and Part B, you notice there is 3 paragraphs side by slide. However, prior to this happening it wasn't side by side, instead it was constantly overlapping each other. I kept trying to put it in container-fluid, rows, and etc. However that didn't really help out. Instead it reminded me of something. At the start of the bootstrap lesson, I learned that an basic screen size there is 12 grids spreaded evenly across. So that means if I have 3 paragraphs and wanted them to be side by side, that would mean I just have to make them take up 4 space each paragraph, and so I did. I applied that to both Part A and Part B. Honestly this was a life saver, because if I didn't learn about bootstrap or think of this, honestly I would of showcased my website, with overlapping paragraphs instead. Another challenge I face was with the image proportion specfically for my carousel. The reason is because when I applied the carosuel on to my website, and put the images, when I refreshed the page, the image were so large that it basically takes up 100% of the screen. At first I didn't know what to do. But then I turned to width and height. I changed both height and width but that didn't seemed to work. So instead of changing both, I changed only width to 85% so that it wouldn't take up 100% of the website size.
