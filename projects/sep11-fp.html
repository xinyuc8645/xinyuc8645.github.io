Product of Our Game

Alright so, as previously mentioned, you should already know what lead to my website of animation. So for this section I'll explained what I put in there and how I created it. At first, I was going to use an bootstrap template to start my website, however there was an major issue. The main issue being that with free bootstrap templates come with code you don't know. Why is that an issue you may ask? Well, if you try changing it, it will be difficult since you don't know what codes does what, and there would be an higher chance that you would mess up your website and the coding as a whole. So I resorted to start from scratch. I first started off my website with the background colors. I usually would go for 1 color background, however thinking that this was the last project for this year I decided to do something special I decided that there should be multiple color at once. More specifically and referring to linear-gradient. If you don't know what that is, basically from one color to another. So I had my color for my background from white--> to blue--> then to dark blue-->. And then my text color would be white, since that's the only color that compliments well with my background. After finishing that I put in an navbar.

The navbar was essential and it was nothing special. I just made the navbar to be fixed-top and the background color to have an blue like color to fit the theme of my website. And after setting all that up. It's time to put in all my information into my website. By far, this was the most easily thing do to. Because all I had to do with copy and paste, since I had the information beforehand. After doing that I just have to create each section for what part. For example, top: the introduction about animation. Underneath that, part A; talks about the software(Adobe), beneath that part B; talks about hardwares, and below that, carousel referring to the hardwares, and below that is some aspect of the aframe tool along side with some pictures, and bottom of the website faq section. There were some bootstrap component I used, for example for the FAQ section, I used accordion, navbar that is responsive, carousel for images, card components to fit the information/images. And that's basically all of my website.
