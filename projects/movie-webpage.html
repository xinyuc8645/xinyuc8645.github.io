Movie Project

Context

For this project, I was pair up with an partner in order to create an prototype, movie webpage that uses our knowledge based on CSS and HTML. The project wasn't hard nor easy, but it was at the right amount of diffculty for me to handle. So is my partner since he's well knowledge in the field of technologies and coding. We deiced to make an mysterious horror movie. And the name would be "Menance"

Content's

In order to first create our movie prototype webpage, me and my partner first have to make a plan first. And in the pickcode website, we created a section specfically for planning called "plan.md" Inside the plan.md section, it includes CSS and HTML like:
  • Divs

Reflection

Challenges

There was several challenges me and my partner come across, while trying to do this project. Including trying to figure out which color for the text would fit the best for the background of our webpage and how to make our text centered, left spaced more and many more. But the one that I specfically struggled on was probably having to create span and divs. The reason is that, I didn't know the difference between the two, but I think mainly divs is for longer stuff to be put inside. Whereas span, is for shorter things. However the main problem was that I didn't know how to really add span and div class, even though we learn this in HTML section long time ago.. Basically, I wanted to make the h2 with different color of the text, size, font and other things. But I didn't know how to do that, because if I just do body or background it would just change the whole thing. So I asked my partner Luka who was well diversed in the fields of technologies and he showed the me proper code. He wrapped the h2 first in div element and then added a class to it calling "story". And then when we go into the CSS section I tryed .story {} and it worked with several elements in there like color, fonts and etc.

TakeAways

The main code I added and to change the h2 of the script of the story was ''' .story { letter-spacing: 6px; color: rgb(178, 181, 181); font-size: 20px; left: 30px; '''
